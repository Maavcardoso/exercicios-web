Dicas Legais

  • Para criar uma tag com div definida, podemos escrever tag#nomedatag. Assim, ele já criará a tag com a div sem a necessidade de digitar mais códigos. Ex: div#noticias = <'div id="noticias"><'/div>
  • Para criar uma tag com classe CSS definida, podemos escrever tag.nomedaclasse. Ex: span.teste = <'span class="teste"><'/span>
